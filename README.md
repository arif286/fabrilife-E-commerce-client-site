### About projects and features

- A customer can choose their products and place order.
- To place order they need to logging first.
- For logging system firebase authentication is implemented.
- After purchase a product user can see their product delivery time schedule.
- order products details will be save in database.
- There have a secure admin panel. Only an admin can manage website, add products delete products, update products and manage customer orders.
